# CW2 Status Review

## Current Position

The project is now beyond a frontend prototype. It includes:

- a working user authentication backend
- route protection
- persistent storage
- REST API routes
- post feed retrieval
- role-aware post publishing
- live comment submission
- Supabase-backed posts, comments, ratings, and image storage

That means the coursework is on track for the core implementation side.

## What Is Strong

- Clear separation between frontend UI and backend route handlers
- Practical cloud-ready structure using Next.js server routes
- Authentication and session management are working
- Persistence exists instead of only mock client-side state
- Social features now demonstrate a more complete use case

## What Is Weak / Incomplete

- No formal automated test suite yet
- No deployment evidence yet
- No slide deck or submission narrative yet
- Authorization is present, but still simple rather than enterprise-grade

## Honest Coursework Position

The main technical gaps are:

- tests
- architecture justification
- presentation/demo assets

So the project is not fully complete yet, but it is in a much better position and covers most of the practical application build.
