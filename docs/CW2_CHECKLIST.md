# CW2 Checklist

## Implemented

- Frontend web application built with Next.js App Router
- User signup, login, logout, and session cookies
- Protected routes for authenticated users
- Role capture for `creator` and `consumer`
- Role-based authorization for publishing posts
- REST-style backend endpoints for auth, posts, and comments
- Persistent data storage using Supabase tables
- Image uploads using Supabase Storage
- Social feed UI with stories carousel, post cards, owner identity, comments, and interactions
- Creator post publishing from the live dashboard
- Consumer/creator separation in the dashboard experience
